Video frame access via Mapping interface.

class mixing.video.video_files.VideoFrames(video_src: str)[source]

Mapping interface to access video frames by index.

Provides dictionary-like access to video frames with support for negative indexing and slicing. Frames are returned as numpy arrays (BGR format).

Parameters:

video_src – Path to the video file

Examples

>>> import tempfile
>>> # Create a small test video (requires actual video file to test)
>>> vf = VideoFrames("test_video.mp4")  
>>> frame = vf[0]  # Get first frame  
>>> last_frame = vf[-1]  # Get last frame  
>>> frames = list(vf[10:20])  # Get frames 10-19
mixing.video.video_files.save_frame(video_src: str, frame_idx: int = 0, *, saveas: str | None = None, image_format: str = 'png')[source]

Extract and save a frame from a video file.

Parameters:

  • video_src – Path to the video file

  • frame_idx – Index of the frame to extract (default: 0)

  • saveas – Output path for the image. Can be: - None: Auto-generate path based on video filename - Path starting with ‘.’: Use as extension (e.g., ‘.jpg’) - Full filepath: Use as-is

  • image_format – Default image format if not specified in saveas (default: ‘png’)

Returns:

Path to the saved image file

Examples

>>> save_frame("video.mp4")  # Saves as video_0.png  
>>> save_frame("video.mp4", 10, saveas=".jpg")  # Saves as video_10.jpg  
>>> save_frame("video.mp4", -1, saveas="last_frame.png")
