mixing.util.to_seconds(value: float, *, unit: Literal['seconds', 'frames', 'milliseconds', 'samples'], rate: float) → float[source]

Convert time value to seconds based on unit.

Parameters:

  • value – Time value to convert

  • unit – Unit of the value (‘seconds’, ‘frames’, ‘milliseconds’, ‘samples’)

  • rate – Frame rate (fps) for ‘frames’ or sample rate (Hz) for ‘samples’

Returns:

Time in seconds

Examples

>>> to_seconds(10, unit="seconds", rate=24)  
10.0
>>> to_seconds(240, unit="frames", rate=24)  
10.0
>>> to_seconds(10000, unit="milliseconds", rate=24)  
10.0
