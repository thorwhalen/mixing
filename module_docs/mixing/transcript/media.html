mixing.transcript.media

ffmpeg-backed audio extraction and timeline-cut application.

These functions shell out to ffmpeg (which must be on PATH); they have no Python dependencies beyond the stdlib.

mixing.transcript.media.apply_keeps(input_path: str | Path, output_path: str | Path, keeps: Sequence[dict], *, video_codec: str = 'libx264', audio_codec: str = 'aac', crf: int = 20, preset: str = 'medium', audio_bitrate: str = '160k', faststart: bool = True, overwrite: bool = True) → Path[source]

Re-encode input_path keeping only the listed time ranges.

Parameters:

  • input_path – Input audio/video file.

  • output_path – Output file (extension picks the container).

  • keeps – [{"start": float, "end": float}, ...] time ranges in seconds.

  • audio_bitrate (video_codec / audio_codec / crf / preset /) – Re-encode knobs forwarded to ffmpeg.

  • faststart – Add -movflags +faststart (mp4 web-streaming).

  • overwrite – Pass -y to ffmpeg.

Returns:

The output path.
