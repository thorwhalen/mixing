mixing.transcript.fillers

Filler-word detection over Scribe-shaped word lists.

A “word list” here is the words array in the ElevenLabs Scribe JSON response — each entry has text, start, end, and type (one of "word", "spacing", "audio_event").

The two main outputs are: - build_cuts(): time ranges to REMOVE (fillers + optional audio events). - keeps_from_cuts(): complementary KEEP ranges, suitable for ffmpeg.

mixing.transcript.fillers.DEFAULT_AUDIO_EVENTS_TO_CUT: frozenset[str] = frozenset({'(coughs)'})

Default audio-event tags to remove. (laughs) is intentionally kept.

mixing.transcript.fillers.DEFAULT_FILLER_TOKENS: frozenset[str] = frozenset({'ah', 'eh', 'er', 'erm', 'hmm', 'mhm', 'mm', 'mmm', 'uh', 'uhh', 'uhm', 'um', 'umm'})

Default filler tokens (lowercased, alpha-only, see normalize_token()).

mixing.transcript.fillers.build_cuts(words: Sequence[dict], *, fillers: Iterable[str] = frozenset({'ah', 'eh', 'er', 'erm', 'hmm', 'mhm', 'mm', 'mmm', 'uh', 'uhh', 'uhm', 'um', 'umm'}), audio_events: Iterable[str] = frozenset({'(coughs)'}), absorb_trailing_space: bool = True, merge_gap: float = 0.08) → list[dict][source]

Compute time ranges to REMOVE.

Parameters:

  • words – Scribe words array.

  • fillers – Override the default filler set.

  • audio_events – Override the default audio-event set.

  • absorb_trailing_space – Extend each cut to the end of the trailing "spacing" token, which avoids leaving a stranded pause.

  • merge_gap – Merge adjacent cuts when their gap is below this many seconds.

Returns:

List of {"start": float, "end": float, "label": str} dicts, sorted by start time and non-overlapping.
