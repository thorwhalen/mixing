mixing.transcript.formats.words_to_prose(words: Sequence[dict], *, paragraph_pause: float = 1.2, drop_fillers: bool = False, fillers: Iterable[str] = frozenset({'ah', 'eh', 'er', 'erm', 'hmm', 'mhm', 'mm', 'mmm', 'uh', 'uhh', 'uhm', 'um', 'umm'}), audio_events: Iterable[str] = frozenset({'(coughs)'})) → str[source]

Render a Scribe word list as plain prose, with paragraph breaks on long pauses.

Parameters:

  • words – Scribe words array.

  • paragraph_pause – Insert a blank line when the gap between two consecutive non-filler words exceeds this many seconds.

  • drop_fillers – If True, omit filler words and removable audio events.

  • audio_events (fillers /) – Override the default filler / event sets.

mixing.transcript.formats.words_to_srt_remapped(words: Sequence[dict], cuts: Sequence[dict], *, max_chars: int = 80, sentence_endings: str = '.?!', drop_fillers: bool = True, fillers: Iterable[str] = frozenset({'ah', 'eh', 'er', 'erm', 'hmm', 'mhm', 'mm', 'mmm', 'uh', 'uhh', 'uhm', 'um', 'umm'}), audio_events: Iterable[str] = frozenset({'(coughs)'})) → str[source]

SRT aligned to a post-cut timeline.

Each word’s timestamp is shifted earlier by the cumulative duration of all cuts that ended before it, so the resulting SRT drops in over the cleaned media.
